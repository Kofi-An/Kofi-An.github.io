Projects

Applied finance and data science projects built on real-world datasets, each one designed to solve an actual business problem, not just demonstrate a technique.

Credit Risk Scorecard

A Basel II compliant probability-of-default model built on 2 million real LendingClub loan records. Uses Weight of Evidence, Information Value feature selection, and logistic regression, validated with Gini, KS statistic, and AUC. Reduces projected portfolio default loss by 18%.

Stack: Python · scikit-learn · Plotly · Jupyter Notebook

Portfolio Risk Dashboard

An interactive live tool that calculates VaR, CVaR, Sharpe ratio, Sortino ratio, and maximum drawdown for any equity portfolio. Paste any stock tickers and get instant risk metrics used by institutional portfolio managers, deployed as a public Streamlit app.

Stack: Python · Streamlit · yfinance · Monte Carlo simulation · Plotly

Financial Fraud Detection

An XGBoost model trained on 284,000 real credit card transactions (0.17% fraud rate). The core challenge, and the real skill is handling severe class imbalance using SMOTE and cost-sensitive learning. Achieves 91% recall while keeping false positives under 5%. SHAP plots explain every prediction.

Stack: Python · XGBoost · SMOTE · SHAP · scikit-learn

  • View on GitHub

Macroeconomic Intelligence Dashboard

A live macro surveillance dashboard pulling from the FRED API and World Bank API, tracking GDP, CPI inflation, unemployment, interest rates, and yield curves across 10 countries including Ghana and major emerging markets. Published on Tableau Public.

Stack: Python · FRED API · World Bank API · Tableau Public · pandas
